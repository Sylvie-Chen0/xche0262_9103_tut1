## Part 1: Imaging Technique Inspiration

### Selected Work

**Abstract Rhythms** by Rosie Bowers  
[Watch the video on Vimeo](https://vimeo.com/440994731)

### Imaging Technique Discussion

**Abstract Rhythms** interests me because it treats abstraction as a living composition, where circles, lines, colour blocks, and empty space keep reorganising through rhythm. The work feels ordered without becoming static: forms slide forward, recede, overlap, blur, and return as new visual structures. I want to borrow this sense of controlled transformation, using simple abstract elements to create visual tension, balance, and harmony. This technique suits the assignment because the image can be driven by time, with each event gradually changing the spatial relationship on screen.

## Part 2: Coding Technique Exploration

### Selected Coding Technique

**Scale transformation in p5.js**  
[p5.js Example: Scale](https://p5js.org/examples/transformation-scale/)

### How this technique could help

Scale transformation in p5.js could help me create the effect of abstract forms moving forward, shrinking into the background, and making space for new patterns. By changing the scale value over time with `frameCount`, circular forms can appear to pulse, recede, or transition between visual layers. This supports the feeling of rhythmic progression in **Abstract Rhythms**, where the composition doesn’t simply move sideways but changes its spatial depth. It also fits the assignment’s time-based mechanic because each visual change can be triggered gradually through timers and repeated events.
